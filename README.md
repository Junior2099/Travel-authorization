# Autorização de Viagem para Criança

Formulário web para geração de Autorização de Viagem Nacional para crianças acompanhadas de pessoa maior, conforme Lei 8069/90 (Estatuto da Criança e do Adolescente - ECA).

## Funcionalidades

- Preenchimento de todos os dados necessários para a autorização
- Geração de PDF pronto para impressão
- Salvamento e restauração de dados em formato JSON
- Máscaras automáticas para CPF, telefone e identidade
- Design responsivo para uso em dispositivos móveis

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Preencha todos os campos do formulário
3. Clique em "Gerar PDF" para baixar o documento pronto para impressão
4. (Opcional) Clique em "Salvar Dados" para guardar os dados em JSON e reutilizar depois

## Base Legal

Artigo 83, § 1º, b, 2, da Lei 8069, de 13 de julho de 1990 - Estatuto da Criança e do Adolescente (ECA).

## Observações Importantes

- A autorização deve ter firma reconhecida em cartório (por semelhança ou autenticidade)
- Apresentar Certidão de Nascimento da criança ou Documento de Identidade com foto
